Welcome to JFire Eclipse Client

This is a demo release of JFire.

Select Open Perspective from the Window menu to see the list of all available perspectives.

For a quick start, try one of those:

  • Overview to quickly find customers, orders, offers, invoices and more.
  • Trade to sell products.
  • Trade administration to create and configure products.
  • Reporting to view reports and their layouts.
  • Reporting administration to manage reports and design report layouts.
  • System administration to manage users, user-groups, access rights and user-settings.
